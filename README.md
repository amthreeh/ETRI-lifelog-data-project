**1. 사용한 데이터** : ETRI 라이프로그 데이터셋 (2020-2018)

- 2020 수면관련 설문결과 데이터
- 2019-2018 수면 측정 데이터
- 2020 수면 측정 데이터

- total1819.csv : 2018,2019년도 수면 관련 설문조사 데이터 + 수면 측정 데이터의 전처리 데이터
